# [registro-diario.md] - DEPRECIACIÓN Y AMORTIZACIÓN ACUMULADA (CÓDIGO 1592 / 1790)

Estas cuentas registran el desgaste, la obsolescencia o el agotamiento de los activos duraderos. Son de **Naturaleza CRÉDITO (-)**.

## 1. Diferencia Fundamental (Base UP + Global)

| Término | Se aplica a... | Ejemplo |
| :--- | :--- | :--- |
| **Depreciación** | Activos Tangibles (Se tocan) | Edificios, Equipos, Maquinaria. |
| **Amortización** | Activos Intangibles y Diferidos | Software, Mejoras a Local, Seguros. |

## 2. El Consejo del "Profe": Cuentas Individuales
Como dice el profesor, para que el **REGISTRO DIARIO** sea transparente, NO debemos mezclar todo en una sola bolsa. Debemos tener:
* `159205` Depreciación Acumulada de Edificios.
* `159210` Depreciación Acumulada de Maquinaria.
* `159215` Depreciación Acumulada de Equipo de Cómputo.

## 3. ¿Cómo funciona en el BALANCE GENERAL?
Se presenta restando directamente al activo para obtener el **Valor Neto en Libros**:
> **Activo Bruto (Costo)** - **Depreciación Acumulada** = **Valor Real Actual**

## 4. Dinámica en el REGISTRO DIARIO (Mensual)
Cada mes, sin que salga un solo centavo del banco, hacemos este movimiento:
* **DÉBITO (+):** Gasto por Depreciación (Cuenta 5160) -> *Baja la utilidad para pagar menos impuestos.*
* **CRÉDITO (-):** Depreciación Acumulada (Cuenta 1592) -> *Baja el valor del activo.*
