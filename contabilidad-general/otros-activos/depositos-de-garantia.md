# [registro-diario.md] - DEPÓSITOS DE GARANTÍA (CÓDIGO 1905)

Identifica los desembolsos de efectivo entregados a terceros para garantizar el cumplimiento de obligaciones contractuales o la prestación de servicios.

## 1. Ejemplos de Aplicación (Base UP + Global)
Para que el sistema de "el lento" sea exacto, registramos aquí:
* **Alquiler de Local:** El depósito (usualmente uno o dos meses) que queda en manos del arrendador o del Ministerio de Vivienda (según el país).
* **Servicios Públicos:** Depósitos iniciales para activar contratos de **Electricidad, Agua o Gas**.
* **Telecomunicaciones:** Garantías por equipos o contratos de **Servicio Telefónico e Internet**.
* **Aduanas:** Depósitos temporales para sacar mercancía mientras se liquidan impuestos.

## 2. Naturaleza y Clasificación
* **Naturaleza:** DÉBITO (+).
* **Clasificación:** Generalmente es un **Activo No Corriente**, porque ese dinero no se recuperará en menos de un año (se recupera cuando se cancela el contrato, que suele ser a largo plazo).

## 3. Dinámica en el REGISTRO DIARIO
* **Aumenta (DÉBITO +):** Cuando entregas el dinero para firmar el contrato.
* **Disminuye (CRÉDITO -):** 1. Cuando te devuelven el efectivo al finalizar el contrato.
    2. Si el proveedor usa el depósito para cobrar una deuda o daño (en ese caso, se convierte en **GASTO**).
